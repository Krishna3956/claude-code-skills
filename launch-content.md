"Claude Code is the best AI coding tool right now."

I keep hearing this from developers. So I started asking a simple follow-up:

"Cool. What are hooks?"

Blank stare.

"What about subagents?"

"...sub what?"

"CLAUDE.md? MCP servers? Slash commands?"

Nothing.

This kept happening. Developers who use Claude Code every single day — shipping production code with it — but running it like a fancy autocomplete. They're using maybe 15-20% of what this thing can actually do.

So I started digging into which features most people miss:

→ Hooks let you trigger automatic actions every time Claude edits a file. Auto-format, auto-lint, auto-test. Most people don't even know this exists.

→ Subagents can run tasks in parallel. You can have Claude write code AND write tests at the same time. Almost nobody uses this.

→ CLAUDE.md is persistent memory. Write your project context once, Claude remembers it forever. Game changer for large codebases.

→ MCP servers are basically plugins. Connect Claude to your database, Figma, APIs — anything. This alone makes it 10x more powerful.

The gap between "I use Claude Code" and "I actually know Claude Code" is massive.

## Tagline (60 chars max)
A 3-minute quiz that tests how well you know Claude Code

## One-liner description
6 rounds, 15 challenges, no coding required — find out if you're a Claude Whisperer or a Terminal Tourist

## Description

### What is it?
"How well do you know Claude Code?" is an interactive quiz that tests your knowledge of Claude Code — Anthropic's AI coding assistant that's been blowing up in the developer community.

### How does it work?
You answer 15 challenges across 6 rounds, each with a different game format:

🔮 **Truth or Myth** — Is this Claude Code feature real or made up?
⚡ **This or That** — Pick how something actually works
🎯 **Quick Pick** — Fill in the blank
⏱️ **Speed Pick** — Tap all correct answers before time runs out
🔍 **Odd One Out** — Four options, one doesn't belong

The whole thing takes about 3 minutes. No sign up. No email. No coding.

### What do you get?
A scorecard out of 100 with a spider-web radar chart showing how you did across 7 skill areas:
- Memory & Context
- Orchestration
- Automation
- Extensibility
- Workflows
- Prompting
- Best Practices

Plus an archetype title based on your score — from "Claude Whisperer" (90+) to "Fresh Install" (you've got some learning to do).

Save your scorecard as an image. Share it on LinkedIn or X with one click (pre-filled share text included).

### Why does it exist?
Everyone's talking about Claude Code but most developers have barely scratched the surface of what it can do. This quiz surfaces hidden features — hooks, subagents, CLAUDE.md, MCP servers, slash commands — in a fun, shareable format.

When someone posts their scorecard, their friends see it and want to try it too. That's the loop.

### The vibe
Clean, minimal design. Cream background, warm orange accents, premium typography. Works great on both mobile and desktop. Designed to feel like something from Anthropic's own design language.
